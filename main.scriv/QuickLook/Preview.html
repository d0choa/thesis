      • Co-evolution-based methods for protein interaction prediction
        This family of methods compares phylogenetic trees in order to predict protein-protein interactions. The tree generation procedure varies depending on the setup of the experiment. Instead of using the trees themselves for comparison, distance matrices are calculated by adding the branch lengths separating each pair of proteins in the tree. The distance matrices are used as input in the following methodologies.

          The original *mirrortree* (MT) approach [#Pazos:2001rm]([](#computmethods.pdf)) calculates the similarity between two distances matrices by calculating the linear correlation coefficient between the corresponding values according to the standard equation:
